# 🐟 FishHunter

## Description
FishHunter is a simple 2D jump-and-run game developed with **Godot**.

This project is based on the following tutorial and extends it with additional gameplay elements:

- Godot Tutorial:  
  https://www.youtube.com/watch?v=LOhfqjmasi0

The game was developed as part of a **game programming lecture project**.

You play as a penguin who must collect all the fish in the area to feed its children.

### Controls
- **Move left/right:** Arrow keys or **A / D**
- **Jump:** Spacebar

### Objective
- Collect all fish in the level
- Avoid other animals to survive
